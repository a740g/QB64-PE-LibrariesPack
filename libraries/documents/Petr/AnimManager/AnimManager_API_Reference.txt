Animation Library – Commands and User Guide

This document is based on the library source package. It focuses on the public commands intended for normal use. The recommended API is the unified interface with the prefix Anim.... Backend APIs (Apng..., Gif89a..., Flic..., AmgAnim..., Ani...) are listed later as an advanced option.

1. What users should normally include

The recommended way is to use the unified manager:

$Include:'anim_manager.bi'

' ... your program ...

$Include:'anim_manager.bm'

On Linux and any other case-sensitive filesystem, use the exact file names:
- anim_manager.bi
- anim_manager.bm
- and all backend files included by the manager

2. Formats currently supported by the manager

The following formats are recognized and usable through the manager:
- PNG and APNG through the APNG backend
- GIF89a
- FLI
- FLC
- Amiga ANIM (FORM ... ANIM)
- Windows ANI (RIFF ... ACON)

Notes:
- A normal PNG is opened through the APNG backend as a 1-frame animation.
- The constant ANIM_FMT_WEBP exists in the source, but it is only reserved. The manager notes that WebP support was removed.

3. Recommended basic workflow

$Include:'anim_manager.bi'

Dim animId As Long
Dim quitFlag As Integer

Screen _NewImage(1280, 720, 32)

animId = AnimOpen("Animals.gif")
If animId >= 0 Then
    AnimSetLoop animId, ANIM_LOOP_FOREVER
    AnimStart animId
End If

Do
    If _KeyHit = 27 Then quitFlag = -1

    Cls
    AnimUpdate animId
    AnimDraw 100, 80, animId

    _Display
    _Limit 60
Loop Until quitFlag

AnimFree animId
End

$Include:'anim_manager.bm'

4. Return value conventions

The library does not use exactly the same style everywhere, but in practice this is what you should expect:

- AnimOpen... returns an ID >= 0 on success, otherwise -1.
- Logical functions such as AnimValid, AnimIsPlaying, and AnimIsCached usually return:
  - -1 = yes / true
  - 0 = no / false
- Action functions such as AnimSeek, AnimSaveFrame, and AnimSaveFrameTo usually return:
  - -1 = success
  - 0 = failure
- Size and state functions such as AnimWidth, AnimHeight, AnimLen, and AnimGetPos return a numeric value. For an invalid ID they usually return 0 or -1 depending on the specific function.

5. Public manager constants

Format constants

ANIM_FMT_NONE = 0
ANIM_FMT_APNG = 1
ANIM_FMT_GIF89A = 2
ANIM_FMT_FLI = 3
ANIM_FMT_FLC = 4
ANIM_FMT_WEBP = 5   ' reserved, WebP support removed
ANIM_FMT_AMIGA_ANIM = 6
ANIM_FMT_ANI = 7

Loop modes

ANIM_LOOP_FILE_DEFAULT = -2
ANIM_LOOP_FOREVER = -1
ANIM_LOOP_ONCE = 0

Meaning:
- ANIM_LOOP_FILE_DEFAULT = follow the format or file default behavior
- ANIM_LOOP_FOREVER = loop forever
- ANIM_LOOP_ONCE = play once and stop

Recommendation: if you want consistent behavior across formats, set looping explicitly with AnimSetLoop.

Cache modes

ANIM_CACHE_STREAM = 0
ANIM_CACHE_PRELOAD_ALL = 1
ANIM_CACHE_WINDOW = 2
ANIM_CACHE_AUTO = 3
ANIM_CACHE_DEFAULT_BUDGET_MB = 256

Practical meaning:
- ANIM_CACHE_STREAM = playback from stream / decoder mode
- ANIM_CACHE_PRELOAD_ALL = try to preload all frames
- ANIM_CACHE_WINDOW = the constant exists, but in the current manager implementation the active mode still falls back to stream
- ANIM_CACHE_AUTO = the manager tries preloading when it makes sense and fits the current limits / budget

6. Public unified Anim... API

This is the main API normal users should use.

Opening and format detection

AnimOpen(fileName As String) As Long
Opens an animation and returns animId.

Example:
animId = AnimOpen("APPLE.FLI")
If animId < 0 Then Print "Open failed"

AnimOpenLimit(fileName As String, maxWidth As Long, maxHeight As Long, maxFrames As Long) As Long
Same as AnimOpen, but checks limits before opening.

Example:
animId = AnimOpenLimit("Animals.gif", 1920, 1080, 500)

If the file is larger or longer than your limit, opening fails.

AnimProbe(fileName As String, ok As Integer, formatId As Long, widthPx As Long, heightPx As Long, frameCount As Long)
Lightweight probe without keeping the file open long-term. Useful for menus, browsers, and validation.

Example:
Dim ok As Integer
Dim fmt As Long, w As Long, h As Long, cnt As Long

AnimProbe "Biker.png", ok, fmt, w, h, cnt
If ok Then
    Print "Format="; fmt; " size="; w; "x"; h; " frames="; cnt
End If

Freeing resources

AnimFree(animId As Long)
Frees one animation and its cache.

AnimFreeAll()
Frees all animations opened by the manager.

Recommendation: at program shutdown, AnimFreeAll is the simplest option.

Playback control

AnimStart(animId As Long)
Starts playback.

AnimStop(animId As Long)
Stops playback.

AnimPause(animId As Long)
Pauses without losing the current position.

AnimResume(animId As Long)
Resumes after pause.

AnimSetLoop(animId As Long, loopMode As Long)
Sets the loop mode.

Example:
AnimSetLoop animId, ANIM_LOOP_FOREVER
AnimStart animId

Timing updates

AnimUpdate(animId As Long)
Updates timing for one animation.

AnimUpdateAll()
Updates timing for all open animations.

Important:
- Without AnimUpdate or AnimUpdateAll, playback will not advance by itself.
- Call it in the main loop.

Drawing

AnimDraw(x As Long, y As Long, animId As Long)
Draws the current frame at native size.

AnimDrawWindow(x1 As Long, y1 As Long, x2 As Long, y2 As Long, animId As Long)
Draws the current frame into the given rectangle.

Important:
- AnimDrawWindow does not preserve aspect ratio. It simply stretches into the box.
- If you want proper fit behavior, calculate it yourself.

Metadata and state

AnimLen(animId As Long) As Long
Returns the number of frames.

AnimWidth(animId As Long) As Long
Returns the width in pixels.

AnimHeight(animId As Long) As Long
Returns the height in pixels.

AnimGetPos(animId As Long) As Long
Returns the current frame index.

AnimIsPlaying(animId As Long) As Integer
Returns -1 if the animation is currently playing.

AnimFormat(animId As Long) As Long
Returns the format identifier ANIM_FMT_....

AnimValid(animId As Long) As Integer
Returns -1 if animId is valid.

AnimError(animId As Long) As String
Returns the last error text for the animation.

Example:
If AnimSeek(animId, 100) = 0 Then
    Print AnimError$(animId)
End If

Direct seeking and stepping

AnimSeek(animId As Long, frameIndex As Long) As Integer
Jumps to a specific frame. The index is clamped to the range 0 .. frameCount - 1.

AnimSeekTime(animId As Long, timeMs As Double) As Integer
Jumps based on time in milliseconds. The manager calculates the matching frame.

AnimStepForward(animId As Long) As Integer
Moves forward by one frame.

AnimStepBackward(animId As Long) As Integer
Moves backward by one frame.

Typical use for manual frame browsing:
If _KeyDown(19712) Then dummy = AnimStepBackward(animId) ' left
If _KeyDown(19200) Then dummy = AnimStepForward(animId)  ' right
AnimDraw 100, 100, animId

Saving a frame to disk

AnimSaveFrame(animId As Long, frameIndex As Long) As Integer
Saves a frame using an automatically generated filename.

The default name looks like this:
originalName_frame_000123.png

AnimSaveFrameTo(animId As Long, frameIndex As Long, fileName As String) As Integer
Saves a frame to the exact file name you specify.

Example:
dummy = AnimSaveFrameTo(animId, 10, "frame10.png")

Cache control

AnimSetCacheMode(animId As Long, cacheMode As Long)
Sets the requested cache mode.

AnimGetCacheMode(animId As Long) As Long
Returns the active cache mode.

AnimSetCacheBudgetMB(mb As Long)
Sets the global cache budget in MB.

AnimGetCacheBudgetMB() As Long
Returns the global cache budget in MB.

AnimIsCached(animId As Long) As Integer
Returns -1 if the animation is using full manager preload cache.

Important detail:
- AnimIsCached = -1 means fully preloaded manager cache.
- It does not mean that some internal cache exists somewhere. A backend may use snapshots or helper buffers and AnimIsCached may still return 0.

Practical use:
AnimSetCacheBudgetMB 512
AnimSetCacheMode animId, ANIM_CACHE_PRELOAD_ALL

7. Recommended usage patterns

7.1 Simple playback of one animation

animId = AnimOpen("3Globes.anim")
If animId >= 0 Then
    AnimSetLoop animId, ANIM_LOOP_FOREVER
    AnimStart animId
End If

Do
    Cls
    AnimUpdate animId
    AnimDraw 50, 50, animId
    _Display
    _Limit 60
Loop Until _KeyHit = 27

7.2 Multiple animations at once

Dim a(1 To 3) As Long

a(1) = AnimOpen("APPLE.FLI")
a(2) = AnimOpen("Animals.gif")
a(3) = AnimOpen("Biker.png")

For i = 1 To 3
    If a(i) >= 0 Then
        AnimSetLoop a(i), ANIM_LOOP_FOREVER
        AnimStart a(i)
    End If
Next i

Do
    Cls
    AnimUpdateAll
    AnimDraw 20, 20, a(1)
    AnimDraw 300, 20, a(2)
    AnimDraw 600, 20, a(3)
    _Display
    _Limit 60
Loop Until _KeyHit = 27

7.3 Manual frame-by-frame browsing

AnimPause animId

If _KeyHit = 19200 Then dummy = AnimStepForward(animId)
If _KeyHit = 19712 Then dummy = AnimStepBackward(animId)

AnimDraw 100, 100, animId

7.4 Exporting one frame

If AnimSaveFrameTo(animId, 0, "preview.png") = 0 Then
    Print "Save failed: "; AnimError$(animId)
End If

8. Notes on cache and smooth playback

This matters especially for heavier formats.

- ANIM_CACHE_STREAM = decode during playback
- ANIM_CACHE_PRELOAD_ALL = load everything into image cache in advance
- ANIM_CACHE_AUTO = let the manager decide
- ANIM_CACHE_WINDOW = in the current manager implementation the active mode still falls back to stream

If you want the smoothest playback and have enough RAM:

AnimSetCacheBudgetMB 512
animId = AnimOpen("3Globes.anim")
AnimSetCacheMode animId, ANIM_CACHE_PRELOAD_ALL
AnimSetLoop animId, ANIM_LOOP_FOREVER
AnimStart animId

10. What is internal and should not be called by normal users

The source contains many helper routines such as:
- AnimTryActivateCache
- AnimPreloadAll
- AnimBackendCloneFrame
- ApngEnsureFrameRendered
- Gif89aLzwDecode
- FlicDecodeChunk...
- AmgAnimApplyDLTA...
- AniDecodeIconChunk

These are not public user commands. They are internal building blocks of the backends and manager.

11. Practical recommendations

- For normal programs, use only the Anim... API.
- If you want consistent behavior, always set:
  - AnimSetLoop
  - and optionally AnimSetCacheMode
- In the main loop, call regularly:
  - AnimUpdate animId
  - or AnimUpdateAll
- At the end, free everything with AnimFree or AnimFreeAll.
- On Linux, always use exact file names, including letter case.

12. Quick cheat sheet

animId = AnimOpen("Animals.gif")
If animId >= 0 Then
    AnimSetLoop animId, ANIM_LOOP_FOREVER
    AnimStart animId
End If

Do
    Cls
    AnimUpdate animId
    AnimDraw 100, 100, animId
    _Display
    _Limit 60
Loop Until _KeyHit = 27

AnimFree animId

Done. This is the layer a normal user of the library should work with.
